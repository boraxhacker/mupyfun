# Overview

Demonstration code to turn on onboard LED when a button is pressed.

## Pins

* GND -> GND
* GP7 -> button 1
* GP6 -> button 2
* GP5 -> uart1 rx
* GP4 -> uart1 tx
* GP16 -> onboard WS2812 RGB LED
